# LAB 4 — Todo CRUD App (React)

Full Create, Read, Update, Delete todo application using React state manipulation.

## Features
- Add new todos with duplicate detection
- Toggle complete / incomplete
- Inline edit (double-click a task, or click ✏️)
- Delete individual todos
- Filter by All / Active / Completed
- Mark all complete / Unmark all
- Clear all completed at once
- Stats bar showing total, active, and done counts

## How to Run
```bash
npm install
npm run dev
```
Open http://localhost:5173
