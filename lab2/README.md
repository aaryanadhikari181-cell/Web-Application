# LAB 2 — Intro to JavaScript

Interactive demo page covering core JavaScript (ES6+) concepts.

## Concepts Covered
1. `var` / `let` / `const` — scoping and declaration
2. Regular functions and default parameters
3. Arrow functions and `this` context
4. Objects — literals, destructuring, methods
5. `Array.map()` — transforming arrays
6. `Array.filter()` — selecting items
7. Spread operator `...` — arrays and objects

## How to Run
Open `index.html` in your browser. Click **"Run All Demos"** to see live output.
