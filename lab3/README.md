# LAB 3 — Login & Register Form (React)

Login and Register forms built with React using a custom reusable `Input` component and React state.

## Features
- Custom `Input` component with label, error, and required-star support
- Controlled inputs with `useState`
- Client-side validation (email format, password length/strength, confirm password match)
- Tab-based view switching between Login and Register
- Success state feedback

## How to Run
```bash
npm install
npm run dev
```
Open http://localhost:5173
